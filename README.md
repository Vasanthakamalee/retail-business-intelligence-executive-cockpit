# Retail Business Intelligence & Executive Decision Cockpit

## Project Overview

Retail organizations generate massive volumes of transactional data but often struggle to transform it into actionable business insights.

This project builds an end-to-end Business Intelligence solution that analyzes retail sales, customer behavior, product performance, and future demand while providing executive-level dashboards for strategic decision-making.

---

## Business Problem

Retail managers often face challenges such as:

- Declining sales without clear root causes
- Customer churn
- Poor inventory planning
- Underperforming products
- Lack of executive reporting

This project addresses these challenges using analytics, machine learning, forecasting, and interactive dashboards.

---

## Objectives

- Analyze sales performance
- Segment customers using RFM Analysis
- Predict customer churn
- Forecast future sales
- Generate executive recommendations
- Build an interactive Power BI dashboard

---

## Dataset

- Online Retail II Dataset
- Over 1 Million retail transactions
